## 🚀 Como Começar

### Opção 1: Visualizar Agora (Sem Setup)
```
1. Abrir arquivo:  c:\Users\User\Documents\GitHub\PJT\index.html
2. Ver no navegador
3. Pronto! ✅
```

### Opção 2: Com Servidor Local
```
1. Instalar Node.js
2. Executar: npm init && npm install express
3. Copiar código de BACKEND-SETUP.html
4. Executar: node server.js
5. Abrir: http://localhost:3000
```

### Opção 3: Deploy Imediato (Recomendado)
```
1. Fazer upload para Vercel/Netlify
2. Website ao vivo em minutos
3. Domínio próprio opcional
```
